uint8_t Neguino::NMCP23S08::getSPIControlByte ( COMMAND  cmd)
protected

returns the SPI control byte

Returns an SPI control byte.

The MCP23S08 is a slave SPI device. The slave address contains four fixed bits (0b0100) and three user-defined hardware address bits (if enabled via IOCON.HAEN; pins A1 and A0) with the read/write command bit filling out the rest of the control byte:: 

+-------------------+
|0|1|0|0|0|A1|A0|R/W|
+-------------------+
|fixed |hw_addr |R/W|
+-------------------+
|7|6|5|4|3|2 |1 | 0 |
+-------------------+
